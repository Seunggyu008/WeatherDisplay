# WeatherDisplay

## 📌 소개 (Introduction)
- 기상청 단기예보 API를 활용한 주변 날씨 검색 앱입니다.

## 🛠️ 기술 스택 (Tech Stack)
- Frontend: JavaScript, Bootstrap
- Backend: ASP.NET Core, C#
- DB: MSSQL, Entity Framework Core

## ⚙️ 기능 설명 (Features)
- 회원가입 / 로그인
- 주변 날씨 정보 조회
- 지역 날씨 정보 조회
- 마이페이지
- 개인정보 수정
